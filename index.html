Sync between devices (Mac ↔ iPad)

Modern iPadOS blocks Safari (and Quick Look) from running JavaScript in local files, so this file can't just be opened from Files on iPad anymore. The fix: host this app shell at a free, stable URL (GitHub Pages works well and costs nothing) and open that URL in Safari like any normal website — no restrictions there. This file ships with no real financial data baked in, on purpose, so it's safe to host publicly. Your actual numbers live only in a private JSON file you load in yourself.

One-time setup: put the hosted app's URL on your Home Screen (Share → Add to Home Screen) on both Mac and iPad — it'll behave like an app icon. Then on each device, first thing: Load from file below, pointing at your private data file (keep that file in iCloud Drive, never in the public hosting).

Ongoing workflow: edit on either device → Save to file → overwrite the copy in iCloud Drive → open the app on the other device → Load from file → pick that same file. Manual, not real-time, but nothing leaves your devices except the empty app shell itself.
